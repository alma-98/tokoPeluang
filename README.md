# TokoPeluang

TokoPeluang adalah platform pencarian peluang bisnis, proyek, kebutuhan vendor, pengadaan, mitra, dan calon klien.

## Project

Project dari Investment Technology Indonesia.

## Fitur

- Cari peluang bisnis
- Filter kategori
- Peluang proyek
- Kebutuhan vendor
- Pengadaan barang dan jasa
- Mitra UMKM
- Kontak langsung melalui SMS dan email
- Responsive mobile dan desktop
